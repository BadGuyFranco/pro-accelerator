# Info-Products Business Model

Marketing framework for courses, coaching, memberships, and digital products.

## Business Model Overview

Info-products deliver transformation through information, frameworks, and implementation support.

### Info-Product Types

| Type | Description | Price Range |
|------|-------------|-------------|
| **Digital Products** | Templates, swipe files, toolkits | $7-$97 |
| **Mini-Courses** | Short, focused training | $47-$297 |
| **Flagship Courses** | Complete transformation system | $497-$2,997 |
| **Group Coaching** | Course + live coaching access | $1,000-$5,000 |
| **Mastermind** | High-touch group experience | $5,000-$50,000 |
| **1:1 Coaching** | Personal transformation | $3,000-$100,000 |
| **Membership** | Ongoing content + community | $27-$297/month |
| **Certification** | Professional credentialing | $2,000-$25,000 |

### The Ascension Ladder

```
FREE: Lead Magnet
   ↓
THRESHOLD ($7-$97): Low-ticket proving you can help
   ↓
CORE ($297-$2,997): Flagship course/program
   ↓
HIGH-TICKET ($3,000-$50,000+): Coaching, mastermind
   ↓
CONTINUITY ($27-$297/month): Membership, community
```

The ladder isn't linear. Customers enter at different points based on awareness and trust.

## Offer Design

Apply all nine components from `Persuasion/Offer-Design.md`:

1. **Core Offer:** Transformation/methodology (not modules)
2. **Deliverables:** Videos, templates, calls, community
3. **Bonuses:** Speed up results, reduce complexity
4. **Value Stack:** Each component with dollar value
5. **Scarcity:** Limited enrollment, cohort-based
6. **Urgency:** Cart close, price increase
7. **Guarantee:** 30-day, conditional, performance-based
8. **Price Anchor:** Compare to alternatives
9. **CTA:** Clear next action

| Audience | Best Funnel | Hook Style |
|----------|-------------|------------|
| **Ready** | Direct offer, tier-ups | Action-focused |
| **Familiar** | Webinar, VSL | Story-based |
| **New** | Lead magnet, Book | Curiosity-driven |

## Presentation Structure

For selling through webinars/VSLs, use the Conversion Theater structure from `Persuasion/Conversion-Experiences.md`:

1. Introduction: Pattern interrupt, promise, credibility
2. Secret 1: Break Method barrier
3. Secret 2: Break Self barrier
4. Secret 3: Break Circumstance barrier
5. Transition: Recap → future pace → offer reveal
6. Stack & Close: Value building → price → guarantee → scarcity → CTA

## Common Playbooks

### Standard Playbook
1. Build audience with opt-in funnel
2. Nurture with Tension Thread + Daily Cadence
3. Launch core offer with webinar
4. Tier-up buyers to coaching
5. Create membership for continuity

### Fast-Start Playbook (New Creator)
1. Create threshold offer ($27-$47)
2. Run to warm traffic
3. Use buyers to fund ads
4. Build webinar for core offer
5. Launch to buyers first

### High-Ticket Playbook
1. Content → Application → Sales call
2. Or: Webinar → Application → Sales call
3. Or: Challenge → Application → Sales call

### Evergreen Playbook
1. VSL or automated webinar
2. Threshold offer tier-ups
3. Automated email sequences
4. Retargeting for non-buyers

## Troubleshooting

| Problem | Likely Cause | Fix |
|---------|--------------|-----|
| Low opt-in | Lead magnet too broad | Make more specific |
| Low show-up | Weak pre-event emails | Strengthen indoctrination |
| Low conversion | Belief barriers unaddressed | Check Method/Self/Circumstance |
| High refund | Overpromise vs delivery | Align offer with capability |
| High churn | Weak onboarding | Improve first experience |

## Quality Checks

- [ ] Prime prospect clearly defined
- [ ] Offer positioned on ladder
- [ ] Clear transformation (not modules)
- [ ] Hook → Story → Offer structure
- [ ] All three belief barriers addressed
- [ ] Value stack with components
- [ ] Guarantee included
- [ ] Real scarcity/urgency
- [ ] Measurement criteria included
- [ ] Follow-up sequences planned
