# Marketing System

Build effective marketing systems by routing to the right frameworks for your specific business model, platform, and goals.

## Quick Start

See `AGENTS.md` for routing logic and usage.

## Structure

| Directory | Purpose |
|-----------|---------|
| `Core/` | Foundational principles (always loaded) |
| `Psychology/` | Attention, trust, belief, value mechanics |
| `Persuasion/` | Hook, story, offer, conversion frameworks |
| `Business-Models/` | Info-products, SaaS, services, creator economy |
| `Funnels/` | Lead capture, presentation, high-ticket, continuity |
| `Sequences/` | Email sequences (onboarding, nurture, launch) |
| `Platforms/` | Platform-specific guidance |
| `Growth-Engines/` | Traffic and growth strategies |
| `Measurement/` | Metrics and optimization |
| `Diagnostics/` | Review checklists and diagnostics |
