# Problem Solver

A structured approach for deep thinking and comprehensive problem solving when tackling complex questions or when standard responses are insufficient.

## Quick Start

When you need deep thinking or comprehensive problem solving, reference the Problem Solver library:

```
/cofounder/tools/Problem Solver/AGENTS.md
```

**Important:** Before using Problem Solver, ask the user if they want to engage the deep thinking process.

## Documentation

- **AGENTS.md** - Complete instructions for AI agents (includes methodology and usage)

## Key Features

- Structured problem-solving methodology - Systematic approach to complex problems
- Multi-perspective analysis - Examines problems from multiple angles
- Deep reasoning framework - Goes beyond surface-level responses
- Thoughtful response generation - Produces comprehensive, well-reasoned answers

## Common Use Cases

Use Problem Solver when:
- User explicitly asks you to "think deeply" about a question or problem
- User expresses frustration with lack of thoughtful responses
- Problem requires comprehensive analysis beyond quick answers
- Complex questions need structured reasoning approach
