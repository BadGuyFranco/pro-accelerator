# Prompt Author

Methodology for writing effective prompts, AGENTS.md files, and instruction documents.

## Documentation

- **AGENTS.md** - Complete methodology for AI agents
- **Prompt Review Process.md** - Workflow for auditing existing prompts

## Key Features

- **Artifact Types** - Standalone prompts, AGENTS.md, library components
- **Writing Process** - Step-by-step prompt creation workflow
- **Composition** - How to structure multi-file prompt systems
- **Review Process** - Triage, stress test, audit, diagnose

## Relationship to Prompt Standards

Core principles (Elegance, Position, XML Boundaries, Quality Checks) live in `.cursor/rules/Prompt Standards.mdc` and auto-load when editing prompts.

This tool provides the **methodology** for complex prompt work. Invoke it when you need to write, review, or debug prompts beyond basic quality checks.
