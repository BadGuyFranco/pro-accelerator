# Local Generator Sub-tool

Offline Word and PDF generation without cloud dependencies.

## Documentation

- **AGENTS.md** - Complete instructions for AI agents

## Key Features

- **Create Word/PDF** - Generate documents from markdown content
- **Read/Extract** - Extract text from existing Word and PDF files
- **Format Conversion** - Convert HTML to markdown, epub, latex via pandoc

## Common Use Cases

Generate documents when cloud services are unavailable. Convert exported Google Docs to markdown or ePub. Extract text from PDF contracts or Word documents.
