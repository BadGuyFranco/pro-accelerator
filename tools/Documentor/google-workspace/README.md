# Google Workspace Sub-tool

Create, edit, read, and export Google Docs, Sheets, and Slides via Google APIs.

## Documentation

- **AGENTS.md** - Complete instructions for AI agents
- **SETUP.md** - Google Cloud Console configuration guide

## Key Features

- **Google Docs** - Create, read, edit, export with markdown support
- **Google Sheets** - Create, read, write spreadsheets
- **Google Slides** - Create presentations, add/remove slides
- **Document Styling** - Control margins, page size, heading styles
- **Collaboration** - Inline markers for comments and suggestions

## Common Use Cases

Create documents directly in Google Drive shared folders. Export Google Docs to Word, PDF, or markdown. Programmatically update spreadsheets. Style documents for print publishing.
